Backdoor Demo (Client-Server)



Учебный проект, демонстрирующий работу потайного хода (backdoor) и методы защиты от него.



Реализована клиент-серверная программа на сокетах с возможностью удалённого выполнения действий на машине клиента.



\## Возможности

\- TCP клиент-серверное взаимодействие  

\- удалённое удаление файла  

\- автоматическое подключение клиента к серверу  

\- автозагрузка клиента в системе  

\- скрытый запуск (без окна)  

\- имитация поведения backdoor  



\## Запуск

1\. Запустить сервер  

2\. Запустить клиент на другой машине (или той же)  

3\. Ввести путь к файлу на сервере  

4\. Файл будет удалён на стороне клиента  



\## Защита

\- анализ открытых портов  

\- блокировка портов через firewall  

\- предотвращение соединения клиента с сервером  



\## Технологии

C / WinAPI / Winsock (TCP sockets)
